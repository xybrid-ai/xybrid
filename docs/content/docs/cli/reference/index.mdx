## Commands

| Command | Description |
|---------|-------------|
| [`run`](/docs/cli/reference/run) | Execute a pipeline |
| [`pack`](/docs/cli/reference/pack) | Package models into `.xyb` bundles |
| [`deploy`](/docs/cli/reference/deploy) | Publish bundles to a registry |
| [`trace`](/docs/cli/reference/trace) | Inspect telemetry from previous runs |

## Quick Examples

```bash
# Run a pipeline
xybrid run --pipeline voice-assistant

# Package a model
xybrid pack whisper-tiny --version 1.0.0 --target macos-arm64

# Deploy to registry
xybrid deploy dist/whisper-tiny.xyb --registry local

# View telemetry
xybrid trace --latest
```
